# wget                                                                                        
                                                                                              
Download files from the Web.                                                                               
                                                                                         
# To download a URL:

```wget <url>```

# To continue an aborted download:

```wget -c <url>```


# Examples:

```
wget https://google.com/index.html

```

# Interview questions:

**Question:** How do you download something?

**Answer:** You run the command `wget` and give it a website to download.
