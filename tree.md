# tree                                                                                        
                                                                                              
Show the contents of the current directory as a tree.                                       
                                                                                              
## Print files and directories up to 'number' levels of depth:
                                                                                              
`tree -L number`                                                                             
                                                                                              
## Print directories only:                                                                     
                                                                                              
`tree -d`                                                                                   
                                                                                              
## Print hidden files too:                                                                     
                                                                                              
`tree -a`
