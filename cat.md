# cat

Print and concatenate/combine files.

# Basic Usage

## show file.txt
    
`cat file.txt`

## show file.txt with line numbers

`cat -n file.txt`                                                                                           

## Concatenate/combine several files into a single file:                                             
                                                                                              
`cat file1 file2 > combined_file `
